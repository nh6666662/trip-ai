## 二、设计规范升级详情

### 2.1 Tailwind 动画 Token（`tailwind.config.ts`）

新增 7 个旅行主题 keyframes 与对应 animation 工具：

| Token | 时长 | 用途 |
|---|---|---|
| `animate-float` | 7s | 光点上下漂浮，营造空气感 |
| `animate-float-slow` | 9s | 慢速光点漂浮 |
| `animate-trace` | 2.4s | SVG 航迹路径描边（forwards） |
| `animate-drift` | 14s | 光点横向漂移渐隐 |
| `animate-shimmer` | 2.5s | 卡片/按钮高光扫过 |
| `animate-breathe-glow` | 3s | CTA 按钮呼吸发光 |
| `animate-skyline-drift` | 16s | 城市天际线轻微浮动 |

### 2.2 全局装饰类（`globals.css`）

新增 9 个旅行主题装饰类：

| 类名 | 作用 |
|---|---|
| `.map-grid` | 浅色地图等高线网格纹理（Hero / Stats 区背景） |
| `.map-grid-dark` | 深色版地图网格（AI 演示区 / 页脚） |
| `.gradient-mesh` | 渐变光晕（CTA / Hero 局部点缀） |
| `.skyline` | 城市天际线剪影（SVG data URI，Hero 底部） |
| `.floating-dots` | 流动光点粒子（::before/::after 伪元素） |
| `.flight-trace` | 飞机航迹 SVG 描边动画基础类 |
| `.route-dashed` | 旅行虚线连接（Feature 区装饰） |
| `.reveal` / `.is-visible` | 滚动浮入工具类（配合 IntersectionObserver） |
| `.text-gradient` | 文字渐变（靛蓝→珊瑚） |
| `.glass-panel` | 面板毛玻璃效果 |

### 2.3 无障碍降级

新增 `@media (prefers-reduced-motion: reduce)` 块，开启系统「减少动态效果」时：
- 所有动画时长降至 0.001ms
- `.reveal` 元素直接显示
- `.flight-trace` 描边直接完成
- `.floating-dots` 伪元素隐藏

## 三、新增共享组件

### 3.1 ScrollReveal / StaggerGroup / StaggerItem

- **ScrollReveal**：基于 framer-motion `whileInView` 的单元素滚动浮入，支持 delay / y / once / amount 参数
- **StaggerGroup**：错峰容器，子元素通过 `staggerChildren` 逐个浮入
- **StaggerItem**：错峰子项
- 统一使用 `cubic-bezier(0.16, 1, 0.3, 1)`（ease-out-expo），与项目总纲 3.8 动效规范一致

### 3.2 CountUp

- 滚动进入视口时数字从 0 计数到目标值
- 基于 IntersectionObserver + requestAnimationFrame
- ease-out-expo 缓动曲线
- 支持 prefix / suffix / decimals
- 尊重 `prefers-reduced-motion`（直接显示终值）

### 3.3 TravelDecorations

| 组件 | 实现 |
|---|---|
| `FlightTrace` | SVG 曲线路径描边动画 + 沿路径运动的飞机（原生 `animateMotion`）+ 起终点标记 |
| `FloatingDots` | 8 个错峰漂浮光点（framer-motion 循环 y/x/opacity 动画） |
| `Skyline` | 城市天际线剪影（CSS 背景图 + 轻微浮动） |
| `RouteLine` | 旅行地标节点 + 虚线连接（SVG） |

## 六、设计原则遵循

本次重设计严格遵循项目总纲设计规范与 frontend-skill / frontend-design 技能指导：

- **克制留白**：每个 section 一个主导视觉、一个动效主题，不堆砌
- **信息降噪**：保留原有文案结构，通过动效增强而非增加信息密度
- **动态呼吸感**：错峰浮入 + 视差 + 计数 + 呼吸发光形成节奏
- **平台原生适配**：Web 端毛玻璃 + framer-motion，保留 Stripe/Linear 质感
- **ease-out-expo 统一**：全站动画使用 `cubic-bezier(0.16, 1, 0.3, 1)`
- **减少动态效果降级**：所有非功能性动画在 `prefers-reduced-motion` 下关闭
- **色彩 Token 引用**：所有颜色使用 primary / accent / neutral Token，无硬编码
- **无通用 AI 美学**：避免紫色渐变白底、Inter 字体堆砌等套路，融入旅行叙事语言
