SECTION 01 · 创意介绍

旅智 TripAI

"把旅行的决策交给 AI，把风景留给你自己。"

生活方式赛道

旅智 TripAI 是一个 AI 驱动的智能旅行规划平台。你在控制台提交一次旅行意图，AI 为你生成紧凑或松弛的行程；旅途中天气、人流、交通的任何变化，都会触发实时预警与备选方案；社区用户的真实反馈持续回流模型，让下一次推荐更懂你。

想解决什么问题：旅行者花大量时间在小红书、马蜂窝、Google Maps 之间反复横跳——查景点、排路线、算时间、订餐厅。好不容易定好的计划，一到现场全乱：突降暴雨、景点排队两小时、餐厅关门、交通延误。规划难、变化多、信息滞后，是旅行三大顽疾。

在一次去杭州的旅行中，早上 9 点到断桥发现人山人海，临时改去苏堤却找不到附近吃饭的地方，下午又突降阵雨打乱了所有户外安排。我问自己：能不能让 AI 像本地朋友那样，提前告诉我什么时候去哪、下雨怎么办、人多了去哪？

— 项目发起人的真实经历

大概是什么产品：Web 控制台 + 桌面应用 + REST API，前后端分离架构。前端 Next.js + React + Framer Motion，后端 Supabase + 高德地图 API + 多 LLM 接入（OpenAI 兼容协议）。AI 生成行程，Supabase Realtime 推送预警，高德提供周边搜索与地理围栏。

一句话概括：旅智 TripAI 不是一个行程生成器，而是一个会"陪你看天气、躲人流、找厕所"的 AI 旅行管家。规划只是起点，陪伴全程才是核心。

SECTION 04 · 核心能力

六项能力，全程陪伴

CORE

AI 行程生成

输入目的地、天数、人数、风格（紧凑/松弛），AI 一键生成可执行行程。支持拖拽调整、节点增删、时间冲突自动检测。生成的不只是地点列表，而是带交通方式、停留时长、备选方案的完整时间轴。

REALTIME

实时预警 + 备选方案

天气、人流、交通三源交叉验证，置信度 ≥ 0.7 推送预警。每个预警附 AI 建议 + 备选方案，一键采纳替换原节点。

AI TIP

AI 关心小贴士

忽略预警后，AI 根据预警内容生成 15 字出行小贴士，写入对应行程节点。施工灰尘→"佩戴口罩防护呼吸道"。

MAP

高德周边搜索

餐厅按餐品类型筛选（川菜/火锅/日料），公共厕所按距离排序，AI 推荐休息点（咖啡馆/商场/公园）。

VISUAL

天气背景动画

根据实时天气动态渲染整个头部卡片背景：晴天太阳光环旋转、雨天雨滴下落、雪天雪花飘落。纯 CSS GPU 加速。
